# nonebot_plugin_chazhanli

适用于 Nonebot2 的 wzry战力查询

##  指令：
以下命令需要加 __命令前缀__（默认为/），可自行设置为空   
 
      /查战力
      /查战力 + 英雄名称 + 游戏战区序号 
      /战力查询
      /战力查询 + 英雄名称 + 游戏战区序号
 
 
   另  __战区序号__ :   
     安卓qq区:*1*  
     安卓wx区:*2*  
     iOSqq区:*3*  
     iOSwx区:*4*
